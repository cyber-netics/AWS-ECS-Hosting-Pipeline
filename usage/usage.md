## Code Base Usage

### Development Environment

- install dependencies for client & backend

```
npm run setup
```

<br/>

- start dev

```
npm run dev
```

<br/>

- [husky](https://typicode.github.io/husky/#/) libary initialize
- it may throw an warning becuse you have already pre-commit files

```
yarn husky init
```
